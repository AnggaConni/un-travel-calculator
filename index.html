New Feature: Deduction Inputs

  • Clarification: The columns "Hotel Days" and "Meal Days" are for DEDUCTIONS.
  • Input Rule: Enter the number of days where meals/hotel were provided FREE by Organizer.
  • If participant pays themselves, type 0 (No deduction).

Deduction Logic

Provided = Free = Deducted from DSA

Hotel Ded = Base * Room% * [Provided Days] Meal Ded = Base * Meal% * [Provided Days]

Floor Rule Policy

  • Incidentals (15-20%) are the staff's minimum entitlement.
  • If active, the system guarantees staff receive at least the Incidental portion even if deductions are high (Full Board).

Terminal Expenses (TF)

  • Standard Round Trip = 4 Legs (Home-Airport-Dest-Airport-Home).
  • Use 0 Legs if an official vehicle is provided (No TF entitlement).
  • Paris HQ has a special TF rate (check ICSC circular).

Advance Payments

Typically, staff receive 80% of estimated DSA before travel.

This calculator shows the 100% Final Entitlement. Deduct any advance received manually during the claim settlement (F10).

Excel Data Format Guide

Required Column Order (Left to Right):

1.Name
2.City
3.Days
4.Hotel Days
5.Meal Days
6.Legs
Column 4: Hotel Days
  • Type Number (e.g., 3) = Accommodation provided FREE by organizer for 3 days (Deduction applies).
  • Type 'Y' or 'Yes' = Provided FREE for ALL days.
  • Type 'N', '0', or blank = Self-paid (No deduction).
Column 5: Meal Days
  • Type Number (e.g., 2) = Meals provided FREE by organizer for 2 days (Deduction applies).
  • Type 'Y' or 'Yes' = Provided FREE for ALL days.
  • Type 'N' or '0' = Self-paid (No deduction).
* Tip: Copy your data from Excel (exclude headers) and paste it into the "Participant Data" box.

Email Reporting Guide

This feature helps you quickly send a summary of the calculation to the finance team or yourself using your default email application.

How it Works

When you click "Launch Mail App", the browser will open your computer's default email client (Outlook, Mail, Gmail, etc.) with the Subject and Body pre-filled with the mission summary.

Audit Trail

If a participant visits multiple locations (multiple rows in data), each entry will be listed separately in the email body, providing a clear audit trail for every leg of the journey.

* Note: If the button doesn't respond, ensure a default email application is configured on your device. For large datasets (>50 rows), please use "Export Excel" instead.
